# MODPACK

This modpack is built to make the transition from 1.8 to modern Minecraft easy, while giving players a solid alternative to using clients. It bundles all the essential mods together and comes preconfigured, so you don’t have to deal with setup or tweaking. Just install and play.

The reason this runs on a modern version instead of 1.8 is simple: there’s basically no downside anymore. You’re not losing anything compared to using a client, but you are gaining access to newer features, better performance options, and more flexibility overall.

This is also what my server runs on, so the goal is to let you switch between competitive 1.8-style gameplay and newer versions without friction. You get the classic feel when you want it, but you’re not locked out of all the newer content and improvements.

# Installation
Install XMCL (or whatever launcher you want): https://xmcl.app/en/guide/install
Download the latest version of the modpack from the **Releases** section.
Click the **Create Game** button to add a new profile.
Click the import from modpack button and select the downloaded mrpack file.
Click Create.
